Modal

Los popups son ventanas emergentes que se abren cuando el usuario interaacciona con cierto elemento de una pagina como un boton o un enlace.
Para que el modal pueda funcionar, Bootstrap lo que hace es usar la etiqueta de data-toggle (en este caso modal) y de la etiqueta de data-tagert(en el ejemplo es el id del modal que se crea abajo). Para cerrar el modal se usa la etiqueta html de data-dismiss="modal" . Dentro del modal puedes colocar el contenido que quieras.
